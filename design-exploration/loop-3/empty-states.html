Design Principles for Empty States

  1. Never a dead end. Every empty state has at least one clear action. Most have two (primary + alternative).
  2. Match the emotional moment. First run = exciting. No search results = helpful, not apologetic. All items checked = celebratory. Error = calm, not alarming.
  3. Use the brand illustration style. Soft circle background + line-weight icon + decorative dots. Consistent across all states. Terracotta for recipe-related, sage for planning/grocery, cream for neutral (search).
  4. Copy is specific, not generic. "No recipes match 'beef wellington'" is better than "No results found." The CTA pre-fills the search term: "Add 'Beef Wellington'" — one less step.
  5. Teach in context. The favorites empty state shows the heart icon and explains where to find it. The first-run state shows the 1-2-3 workflow. Education happens where confusion would otherwise live.
  6. Celebratory empties are a feature. The grocery "all done" state feels rewarding, not blank. Small touches (confetti dots, warm copy) turn a functional moment into an emotional one.
